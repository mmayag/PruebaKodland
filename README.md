# Defensor Eléctrico

## Descripción
Defensor Eléctrico es un juego desarrollado en Python usando la librería PyGame.
El jugador controla un ingeniero eléctrico que debe proteger una estación de energía de sobrecargas.

## Cómo jugar
- Flecha izquierda → Moverse a la izquierda
- Flecha derecha → Moverse a la derecha
- ENTER → Iniciar juego

## Características
- Menú principal
- Sistema de puntuación
- Enemigos aleatorios
- Detección de colisiones
- Funciones personalizadas

## Tecnologías usadas
- Python 
- PyGame
